# 📖 Ruby on Rails - Hello world React Rails app TWO <a name="about-project"></a>

Set up a "Hello world" project with two apps. Now that you're familiar with Rails and React, it's time to put them together in a new kind of 'Hello World!' app. This exercise is going to have you create a React front-end with a Rails back-end and connect them to display a random message.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [Node](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)
- [Ruby](https://www.ruby-lang.org/en/)
- [Rails](https://rubyonrails.org/)
- [PostgreSQL](https://www.postgresql.org/)
- [Git](https://git-scm.com/)

### Setup

Clone this repository to your desired folder:

`git clone https://github.com/Abdulwhbhussain/hello-react-front-end.git`

then

`cd hello-react-front-end`

### Install

Install this project with:

``npm install``

### Usage

`npm start`
